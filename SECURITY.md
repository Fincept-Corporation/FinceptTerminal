# Security Policy

## Reporting a Vulnerability
- Open a private Security Advisory on GitHub
- Or email: security@fincept.in

## Supported Versions
| Version | Supported |
|---------|-----------|
| v4.x    | ✅ |
| v3.x    | ✅ |
| v2.x    | ❌ |

## Security Notice
This application handles financial data. Never commit API keys or credentials.
Use environment variables for all sensitive configuration.
