## 📥 Descargar

**vz. 0,11**| Windows • MacOS • Linux

[![Download](https://img.shields.io/badge/Download-Latest%20Release-brightgreen?style=for-the-badge)](https://github.com/Fincept-Corporation/FinceptTerminal/releases)[![Microsoft Store](https://img.shields.io/badge/Microsoft%20Store-Get%20App-blue?style=for-the-badge&logo=microsoft)](https://apps.microsoft.com/detail/XPDDZR13CXS466?hl=en-US&gl=IN&ocid=pdpshare)

**Construir desde la fuente:**

```bash
git clone https://github.com/Fincept-Corporation/FinceptTerminal.git
cd FinceptTerminal/fincept-terminal-desktop
bun install && bun run tauri:dev
```

**Requisitos:**

-   [Bien](https://bun.sh)1.0+
-   [Óxido](https://www.rust-lang.org/)(último establo)
-   Dependencias específicas de la plataforma (ver[documentos/GETTING_STARTED.md](docs/GETTING_STARTED.md))

### Instalación de macOS (compilaciones sin firmar)

⚠️**Importante:**Las compilaciones de desarrollo y artefactos no están firmadas con un certificado de desarrollador de Apple.

**Después de descargar el DMG:**

```bash
# Remove quarantine flag from DMG
xattr -cr ~/Downloads/FinceptTerminal-*.dmg

# Open DMG, drag to Applications, then remove quarantine from app
xattr -cr /Applications/FinceptTerminal.app
```

**Alternativa:**Haga clic derecho en la aplicación → Seleccione "Abrir" → Haga clic en "Abrir" en el cuadro de diálogo de advertencia.

Comunicados oficiales de la[Página de lanzamientos](https://github.com/Fincept-Corporation/FinceptTerminal/releases)Incluye instrucciones de instalación detalladas.
