## 🎯 Ce qui nous distingue

## 📥 Télécharger

**vz. 0,11**| Windows • MacOS • Linux

[![Download](https://img.shields.io/badge/Download-Latest%20Release-brightgreen?style=for-the-badge)](https://github.com/Fincept-Corporation/FinceptTerminal/releases)[![Microsoft Store](https://img.shields.io/badge/Microsoft%20Store-Get%20App-blue?style=for-the-badge&logo=microsoft)](https://apps.microsoft.com/detail/XPDDZR13CXS466?hl=en-US&gl=IN&ocid=pdpshare)

**Construire à partir de la source :**

```bash
git clone https://github.com/Fincept-Corporation/FinceptTerminal.git
cd FinceptTerminal/fincept-terminal-desktop
bun install && bun run tauri:dev
```

**Exigences:**

-   [Bien](https://bun.sh)1.0+
-   [Rouiller](https://www.rust-lang.org/)(dernière version stable)
-   Dépendances spécifiques à la plateforme (voir[docs/GETTING_STARTED.md](docs/GETTING_STARTED.md))

### Installation de macOS (versions non signées)

⚠️**Important:**Les versions de développement et d'artefact ne sont pas signées avec un certificat de développeur Apple.

**Après avoir téléchargé le DMG :**

```bash
# Remove quarantine flag from DMG
xattr -cr ~/Downloads/FinceptTerminal-*.dmg

# Open DMG, drag to Applications, then remove quarantine from app
xattr -cr /Applications/FinceptTerminal.app
```

**Alternative:**Cliquez avec le bouton droit sur l'application → Sélectionnez « Ouvrir » → Cliquez sur « Ouvrir » dans la boîte de dialogue d'avertissement.

Les communiqués officiels du[Page des versions](https://github.com/Fincept-Corporation/FinceptTerminal/releases)inclure des instructions d’installation détaillées.
