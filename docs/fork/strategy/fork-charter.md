# Fork Charter

Status: Active

## Purpose

This fork exists to support custom product, design, and implementation changes while staying maintainable alongside upstream updates.

## Goals

- Preserve a maintainable upstream sync path.
- Support fork-specific design and code changes.

## Non-Goals

- Avoid unnecessary rewrites of upstream contributor guidance.
- Avoid a heavyweight documentation process for small edits.

## Upstream Alignment Principles

- Keep upstream build, architecture, and contributor guidance intact unless the fork has a clear reason to diverge.
- Prefer fork-specific behavior, branding, and workflow changes to live in fork-owned docs and targeted implementation work rather than broad repo-wide rewrites.
